# ApexFit 

A fitness web app to explore workouts, calculate BMI, and book classes — built with HTML, CSS, and JavaScript.

---
## Features

- **Home** — Hero section with quick-action buttons
- **Workouts** — 8 workout cards with photos, filters, and exercise detail modals
- **BMI Calculator** — Live BMI calculation with a colour-coded result bar
- **Classes** — 7-day timetable with spot availability and booking

---

## How to Run

1. Download `index.html`, `style.css`, and `script.js` into the same folder
2. Double-click `index.html` to open in your browser

> Requires an internet connection for fonts and workout images.
